Países, estados y ciudades

Una librería liviana para Node.js que sirve catálogos geográficos. Carga lazy — solo lee el JSON que pedís, cachea en memoria, y mantiene el cold start en el mínimo. Probala abajo con selectores encadenados.

Ciudades

Cómo se usa en tu código

node.js

1 · Instalación 

npm install ylazzari-countrystatecity

2 · Uso básico 

import {
  CountryService,
  StateService,
  CityService,
  CountryEnum,
} from "ylazzari-countrystatecity";

// Países
const countries = new CountryService().getAllCountries();

// Estados de un país
const states = new StateService()
  .getStatesByCountry(CountryEnum.AR);

// Ciudades de un estado (lazy load + cache)
const cities = new CityService()
  .getCitiesByCountryAndState(CountryEnum.AR, 3430657);

Esta demo corre un mini server Node que importa la librería desde dist/ y expone /api/*.
